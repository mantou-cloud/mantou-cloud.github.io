# Kaiyuan Deng - Personal Website

This repository contains the source code for my personal academic and engineering website:

**[mantou-cloud.github.io](https://mantou-cloud.github.io/)**

I am an M.Eng. student in Computer Technology at the University of Electronic Science and Technology of China (UESTC). My interests include embodied intelligence, multimodal perception, robotic manipulation, and vision-language-action models.

## Website contents

- About, education, and research interests
- Research publications and project contributions
- Engineering experience in robotics and autonomous-driving simulation
- Selected projects with system diagrams and demonstrations
- Downloadable resume

## Selected projects

- Closed-Loop Driving Simulation
- MiVLA on LocoMan
- Koala One Composite Robot
- MagicBot Dual-Engine System
- Forklift Cargo Measurement
- 3D Scene Understanding

## Built with

The website is built with [Jekyll](https://jekyllrb.com/) and the [al-folio](https://github.com/alshedivat/al-folio) theme, and is published through GitHub Pages.
