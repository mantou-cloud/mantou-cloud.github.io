This website presents my background, selected projects, technical interests, and future work in one place.

It is built with **Jekyll** and the open-source **al-folio** theme, with automated deployment through **GitHub Pages**. The site is intentionally kept simple, responsive, and easy to maintain.

## Highlights

- English-first personal and technical profile
- Responsive light and dark themes
- Project and GitHub activity pages
- Privacy-conscious public content
- A structure ready for future publications, notes, and an online CV

[View the source on GitHub](https://github.com/mantou-cloud/mantou-cloud.github.io)
